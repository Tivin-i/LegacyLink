# Contributing to LegacyLink

First off, thank you for considering contributing to **LegacyLink**! As a self-hosted, local-first system, the strength of this project relies on the community making it more resilient and versatile.

## Our Philosophy
LegacyLink is built to ensure that documentation survives and stays accessible for the long term. Because of the sensitive and personal nature of legacy data, we prioritize:
* **Privacy-First:** Local-only by default.
* **Sustainability:** Code that is easy to maintain.
* **Transparency:** Source-available code under the **PolyForm Noncommercial License 1.0.0**.

## How to "Commit Back"
While our license allows you to modify the code for personal and non-commercial use, we strongly encourage you to share those improvements back with the main project.

## Reporting Issues
If you find a bug but don't have a fix yet, please open an **Issue** on GitHub. Include:
* Your operating system and browser/environment.
* Clear steps to reproduce the bug.
* Expected vs. actual behavior.

## Development Setup

1. Clone your fork: `git clone https://github.com/your-username/LegacyLink.git`
2. Install dependencies: `npm install`
3. Run in development mode: `npm run dev`
